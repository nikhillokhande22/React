# Students Table React Project

This is a simple React project that displays a table of students with their details.

## Features
- Display students in a table
- Add new students
- Edit student details
- Delete students
- Clean React UI

## Technologies Used
- React.js
- JavaScript
- HTML
- CSS

## Project Structure
src/
 ├── App.js
 ├── index.js
 ├── components/
public/
package.json
## Installation

1. Clone the repository

git clone https://github.com/nikhillokhande22/React

2. Install dependencies

npm install

3. Run the project

npm start
